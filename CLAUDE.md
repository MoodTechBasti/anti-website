This file provides guidance to Claude Code when working in this repository.

## Project Context
This project is an interactive landing page for MoodTech Solutions, adapting the visual structure and particles/gravity animations of https://antigravity.google/.

## Tooling Pointer
- The machine-wide Windows CLI baseline lives in `C:\Tools\TOOLING.md`.
- Default Python command is `python`.

## Core Commands
- Dev Server: `npm run dev`
- Build: `npm run build`
- Validate: `pwsh C:\Tools\check-tooling-pointer.ps1`
